# Stepper

Подходит в ситуациях, когда нужно явно показать пользователю сколько шагов нужно пройти.
И без предыдущего шага следующий невозможен или не имеет смысла.
Позволяет сфокусировать внимание на текущий шаг и скрыть детали остальных шагов.
